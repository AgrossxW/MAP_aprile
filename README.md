# MAP - Progetto Clustering QT (esercitazione 7)

Progetto Java per l'esame di Metodi Avanzati di Programmazione, consegna 08.04.2026 .

## Requisiti
- Java 8+
- MySQL attivo
- Database MapDB configurato

## Setup Database
Eseguire:

mysql -u root -p < setup_mapdb.sql

## Esecuzione
cd src
./run.sh

## Descrizione
Il progetto implementa l'algoritmo QT per il clustering,
con lettura dati da database MySQL tramite JDBC.
